# Learning Management System (LMS) - MERN Stack

## Project Overview
A comprehensive LMS with role-based access for Admin, Instructor, and Students. Features include course management, user tracking, and secure authentication.

## Technologies Used
- **Frontend**: React.js, Axios, Bootstrap
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **Security**: JWT, Bcrypt, Environment Variables

## Installation Steps
1. Clone the repository.
2. Run `npm install` in both `frontend` and `backend` folders.
3. Add your `MONGO_URI` and `JWT_SECRET` in the `.env` file.
4. Run `npm start` (Frontend) and `node server.js` (Backend).
