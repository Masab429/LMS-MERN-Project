# Learning Management System (LMS) - MERN Stack

## Project Overview
A comprehensive LMS with role-based access for Admin, Instructor, and Students. Features include course management, user tracking, and secure authentication.

## Technologies Used
- **Frontend**: React.js, Axios, Bootstrap
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose)
- **Security**: JWT, Bcrypt, Environment Variables

## Installation Steps

1. **Clone the Project**:
   `git clone https://github.com/Masab429/LMS-MERN-Project.git`

2. **Setup Backend**:
   - Go to `backend` folder: `cd backend`
   - Install dependencies: `npm install`
   - Configure `.env` file with your `MONGO_URI` and `JWT_SECRET`.
   - Start Server: `node server.js`

3. **Setup Frontend**:
   - Go to `frontend` folder: `cd frontend`
   - Install dependencies: `npm install`
   - Start React App: `npm start`

4. **Access the App**:
   Open `http://localhost:3000` in your browser.
